Lista não Ordenadas

voltar

O HTMLsuporta listas ordenadas, não ordenadas e de definições.

Uma lista não ordenada é uma lista de itens.

As listas de itens são marcadas com bullets (tipicamente pequenos círculos pretos).

Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag e termina com uma tag <li>.

Exemplo:

<ul>
<li>Café</li>
<li>Leite</li>
</ul>

Aqui está como aparece em um navegador:

  • Café
  • Leite

Dentro de um item de uma lista você pode colocar parágrafos, quebras de linha, imagens, vínculos, outras listas, etc.

Lista Ordenadas

voltar

Uma lista não ordenada é também uma lista de itens.

As listas de itens são marcadas com números.

Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag e termina com uma tag <li>.

Exemplo:

<ol>
<li>Café</li>
<li>Leite</li>
</ol>

Aqui está como aparece em um navegador:

  1. Café
  2. Leite

Dentro de um item de uma lista você pode colocar parágrafos, quebras de linha, imagens, vínculos, outras listas, etc.

Listas de Definições

voltar

Uma lista de Definiçãos não é uma lista de itens.

Esta é uma lista de termos e explicações dos termos.

Uma lista de definições começa com a tag <dl>. Cada termo da lista de definições começa com a tag <dt>.

Cada definição da lista de definições começa com a tag <dd>.

Exemplo:

<dl>
<dt>Café</dt>
<dd>Bebida quente preta</dd>
<dt>Leite</dt>
<dd>Bebida fria branca</dd>
</dl>

Aqui está como aparece em um navegador:

Café
Bebida quente preta
Leite
Bebida fria branca

Dentro de um item de uma lista você pode colocar parágrafos, quebras de linha, imagens, vínculos, outras listas, etc.
